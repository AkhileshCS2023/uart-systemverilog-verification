# UART SystemVerilog Verification

This project implements verification of a UART design using a SystemVerilog testbench architecture.

## Testbench Architecture
- Generator
- Driver
- Monitor
- Scoreboard
- Reference Model
- Environment
- Functional Coverage

## Tools Used
- QuestaSim

## Features
- Mailbox-based communication
- Functional coverage
- Modular verification architecture
- Interface-based DUT connection
